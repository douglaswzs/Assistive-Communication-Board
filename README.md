# Assistive Communication Board

A web-based assistive communication board for people with aphasia (e.g., after stroke) to communicate basic needs, feelings, pain, and daily routines. Designed for use in care facilities and hospitals, with multi-language and high-contrast accessibility support.

## Features

- **Multi-language support:** English, Mandarin Chinese (简体中文), Cantonese Chinese (繁體中文/香港).
- **Separate Menu and Voice Language:** Menu can be in one language (for staff), speech output in another (for patient).
- **Accessible UI:** High-contrast mode, large buttons, emoji support.
- **Pain scale:** Visual pain scale with color gradient.
- **Admin mode:** Change languages and settings.
- **Daily routine/activities section:** Common everyday interactions.
- **Speech output:** Uses browser speech synthesis in selected language.

## Usage

1. **Open `index.html` in your browser.**
2. **Admin Settings:**  
   - Click ⚙️ Admin to set Menu Language and Voice Language.
   - High-contrast mode toggle available.
3. **Customizing phrases:**  
   - Edit `langMap.json` to add, remove, or change phrases and translations.
   - Each phrase uses a `phraseid` for cross-language mapping.
## Accessibility

- High-contrast mode for low-vision users.
- Large, touch-friendly buttons.
- Emoji and color cues for quick recognition.
